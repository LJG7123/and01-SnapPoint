## 프로젝트 소개
**지도로 확인하는 여행 기록, SnapPoint**
- 내 여행을 게시글 형태로 기록할 수 있어요
- 게시글에 추가한 미디어가 촬영된 위치를 스냅포인트 형태로 저장할 수 있어요
- 미디어들은 경로로 이어져 지도에서 한 눈에 확인할 수 있어요
